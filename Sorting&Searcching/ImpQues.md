## Must-do (very important)

* **Distinct Numbers** — sorting / set
* **Apartments** — two pointers (matching)
* **Ferris Wheel** — greedy + two pointers
* **Concert Tickets** — multiset / binary search
* **Restaurant Customers** — sweep line / events sorting
* **Movie Festival** — interval scheduling greedy
* **Sum of Two Values** — hash map / two pointers
* **Maximum Subarray Sum** — Kadane’s algorithm
* **Missing Coin Sum** — greedy prefix coverage (VERY important)
* **Collecting Numbers** — permutation indexing
* **Traffic Lights** — set + multiset intervals
* **Room Allocation** — interval scheduling + priority queue
* **Factory Machines** — binary search on answer
* **Tasks and Deadlines** — greedy sorting
* **Sum of Three Values** — sorting + two pointers
* **Sum of Four Values** — pair sums / hashing
* **Nearest Smaller Values** — monotonic stack
* **Subarray Sums I** — prefix sum + hashmap
* **Subarray Divisibility** — prefix mod frequency
* **Array Division** — binary search on answer

## Very useful (2-pointer / sliding window mastery)

* **Playlist** — sliding window (distinct elements)
* **Towers** — patience sorting / multiset
* **Subarray Sums II** — prefix sum + hashmap (advanced)
* **Maximum Subarray Sum II** — prefix sum + deque
* **Movie Festival II** — multiset greedy (multi-resource scheduling)

## Lower priority (still good practice)

* Stick Lengths
* Reading Books
* Collecting Numbers II
* Nested Ranges Check / Count
* Josephus I / II

## If preparing for interviews (priority order)

1. Binary search on answer (Factory Machines, Array Division)
2. Two pointers (Apartments, Ferris Wheel, Sum problems)
3. Sliding window (Playlist)
4. Prefix sums + hashmap (Subarray problems)
5. Sweep line / intervals (Room Allocation, Movie Festival)
6. Monotonic stack (Nearest Smaller Values)
7. Greedy coverage (Missing Coin Sum)
